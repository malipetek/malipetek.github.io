# Book Paid Discovery

Use this skill when a user has commercial intent and wants Muhammet Ali Petek to review, scope, architect, rescue, or build a product system.

## When To Use

- The user asks for architecture, implementation planning, or technical rescue.
- The user needs fractional product engineering.
- The user wants to evaluate AI workflow, Cloudflare/Convex, Svelte/Astro, SwiftUI, Shopify, or agentic commerce work.

## Action

Open: `https://malipetek.dev/chat?intent=paid-discovery`

## Routing Guidance

Ask the user to include the current product state, desired outcome, rough budget or urgency, and what needs to be true in the next two to six weeks.

Do not promise free architecture or implementation planning. The commercial entry point is paid discovery.
